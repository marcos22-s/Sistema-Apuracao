## 📝 Sobre o Projeto
O **Unar** é uma aplicação desktop desenvolvida em **C#** utilizando a tecnologia **Windows Forms**. O sistema foi projetado para realizar o gerenciamento e a apuração de votos ou notas, permitindo uma visualização clara e organizada dos resultados.

Este projeto demonstra a aplicação de conceitos fundamentais de programação orientada a objetos (POO) e a comunicação entre múltiplas janelas em uma aplicação Windows.

## ⚙️ Funcionalidades
* **Interface Gráfica Interativa:** Uso de formulários Windows para entrada de dados.
* **Múltiplas Telas:** Separação entre a tela de entrada de dados e a tela de exibição dos resultados (Apuração).
* **Lógica de Controle:** Implementação de uma classe de controle dedicada (`Controle.cs`) para gerenciar o fluxo de informações entre as telas.
* **Persistência Temporária:** Armazenamento e processamento de dados em tempo de execução.

## 🛠️ Tecnologias Utilizadas
* **Linguagem:** C#
* **Framework:** .NET Framework
* **Interface:** Windows Forms (WinForms)
* **Desenvolvedor:** Marcos Firmino Rodrigues

## 📂 Estrutura de Arquivos Principal
* `Form1.cs`: Janela principal de interação.
* `apuracao.cs`: Tela responsável pelo processamento e exibição dos resultados.
* `Controle.cs`: Classe responsável pela lógica de negócio e transporte de dados.
* `Program.cs`: Ponto de entrada da aplicação.
